常用命令

新建草稿

hugo new hello.md

hugo new about/index.md

hugo new post/first.md

生成文章，并启动hugo

$ hugo server -D

$ hugo server -t even –buildDrafts

-t 指定渲染主题， –theme=hyde

-D –buildDrafts 生成标识为草稿的页面，即draft = true的文章也会生成。
