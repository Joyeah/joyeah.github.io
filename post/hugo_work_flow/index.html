hugo托管到github pages完整工作流程

涉及工具

git/TortiseGit

Typora

准备

github.io站点

1.创建github帐号

2.建立以帐号名同名的库

3.配置github.io。你的站点地址就是：username.github.io

下载安装hugo

下载： https://github.com/gohugoio/hugo/releases

官网： https://gohugo.io/getting-started/installing/

注意把hugo所在路径添加到环境变量，这样在任务路径下就可以直接使用hugo了。

开始工作

添加文章

1

# hugo new post/article1.md

编辑文章

可以采用Typora编辑文章内容。

可在文章头部区域添加tag,categories。类似如下：

1
2

tag: ["hugo", "博客"]
categories: ["笔记"]

编辑好后，可以将draft改为false。

本地发布

1
2

# cd <你的本地hugo站点根目录>
# hugo

若把草稿（draft: true）也一同发布，则执行：

1

# hugo -D

本地运行

1
2

# hugo server
# hugo server -D

打开浏览器，输入地址http://localhost:1313/查看

提交文章

1
2
3
4
5

# cd public
# git status (查看变更状态)
# git add .
# git commit -m "add a article"
# git push origin master

可能需要等一会，才能看到你的站点更新了。
